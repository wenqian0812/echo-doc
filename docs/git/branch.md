## GIT分支管理

1. 查看分支

```shell
git branch (branchname)
```

2. 切换分支

```shell
git checkout (branchname)
```

3. 创建分支

```shell
 git branch testing
```

4. 创建分支并且切换到创建分支

```shell
git checkout -b newtest
```

5. 删除分支

```shell
git branch -d (branchname)
```

6. 删除远程分支

```shell
git push origin --delete [branch_name]
```
